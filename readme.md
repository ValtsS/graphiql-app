## Description

GraphiQL is a playground/IDE for graphQL requests.

## Installation

```bash
$ git clone https://github.com/ValtsS/graphiql-app.git
$ cd graphiql-app
$ git checkout develop
$ npm i
$ npm run dev
```

## Running the app

```bash
# development
$ npm run dev

# preview mode
$ npm run preview
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run coverage
```
